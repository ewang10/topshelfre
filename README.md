# topshelfre take home assessment

## Environment
1. IDE of choice (e.g. VSCode)
2. Node v20.10.0
3. NPM 10.2.3

## Instructions
1. Run `npm install` to install dependencies
2. Run `npm start` to start the node server on http://localhost:3000
3. Run `npm test` to run the tests
